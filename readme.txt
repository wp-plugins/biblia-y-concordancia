This plugin installs the Spanish Bible 1960 version and allows the user to search the Bible by book, chapter, verse and words.

== Description ==

Este plugin de WordPress instala la Biblia en español versión Reina Valera 1960 © American Bible Society, y permite
al usuario buscar en la Biblia por medio de libro, capítulo, versículo y concordancia.

== Installation ==

1. Upload the plugin `biblia y concordancia` to the `/wp-content/plugins/` directory
    Sube el plugin `biblia y concordancia` al directorio `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
    Activa el plugin por medio del menú 'Plugins' en WordPress
3. Place the shortcode `[bendicion_biblia]` in a new page
    Agrega el código `[bendicion_biblia]` en una página nueva
4. You may also use this line in your template:
    También puede ser activado al agregar este código en un template:
   `<?php if (function_exists('display_bible_form')) { display_bible_form(); } ?>`
